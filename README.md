## A} ArrayList
1. Dynamic list/  resizable-arrays
2. The order of elements is preserved as Insertion order.
3. You can add null elements.
4. Allows duplicate values.

## B} LinkedList 
1. Doubly Linked List.
2. The order of elements is preserved as Insertion order.
3. Allows Duplicates and Null values.
